# File Format:
# This dictionary file contains multiple lines of records and/or comments.
# Any line starting with a '#' or '<' character is a comment line.
# Each non-comment line contains a record. A record is a (key,value) pair,
# where the key and the value is separated by a space character (0x20).
# A key is a string of characters from the set [A..Z,\].
# A value is a string of double-byte Han characters (hanzi).
# All the hanzi in a record have the same Cantonese transcription.
#
# Caveat:
# Note that some infrequently used characters can be input by typing '\',
# followed by the Cantonese transcription of the respective radical.
# Blank hanzi are due to unsuccessful GB-BIG5 conversion and thus
# should be ignored.
#
