# Format String Syntax

## In Snippets

### Placeholders

	$«int»
	${«int»}
	${«int»:«snippet»}
	${«int»/«regexp»/«format»/«options»}
	${«int»|«choice 1»,…,«choice n»|}     # parsed but not handled

### Code

	`«code»`

## In Format Strings

	$0-n

	\U, \L, \E, \u, \l
	\n, \t

	«variables»

	(?«var»:«if»:«else»}
	(?«var»:«if»}

## In Both

### Variables

	${«var»:?«if»:«else»}
	${«var»:+«if»}
	${«var»:-«else»}
	${«var»:«else»}
	${«var»/«regexp»/«format»/«options»}
	${«var»:[/upcase][/downcase][/capitalize][/asciify]}
